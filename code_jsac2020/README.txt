*********************************************************************************************************************************
This simulation code package is mainly used to reproduce the results of the following paper [1]:

[1] M. Ke, Z. Gao, Y. Wu, X. Gao, and K-K Wong , “Massive Access in Cell-Free Massive MIMO-Based Internet of Things: Cloud Computing and Edge Computing Paradigms,” to appear in IEEE Journal on Selected Areas in Communications.

If you use this simulation code package in any way, please cite the original paper [1] above. 

The author in charge of this simulation code pacakge is: Malong Ke (email: kemalong@bit.edu.cn).



*********************************************************************************************************************************
How to use this simulation code package?

1. Run the file named "main_massive_access.m" to get the performance data of the corresponding scheme. 

2. The System Parameters in the file "main_massive_access.m" could be changed according to your demands.
